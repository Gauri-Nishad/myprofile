Project 1: Agriculture Land Rental Platform

This platform is designed for farmers and individuals interested in agriculture and gardening. It allows landlords to list suitable agricultural land for tenants to rent. The platform ensures only authorized lands are available for renting, and provides a contact system for easy communication between tenants and landlords.

  • Developed using HTML for structuring web pages, CSS for styling, and JavaScript for dynamic content.
  • Utilized Python and Flask, a lightweight web framework, for rapid application development.
  • Implemented features for landlords to manage land listings and for tenants to search and contact landlords.
  • Ensured the platform supports only agricultural purposes for listed lands.

Technologies Used: HTML, CSS, JavaScript, Python, Flask.

Project 2: E-commerce Web Application for Dairy Products

Developed an e-commerce web application using Django for selling various types of dairy products. The platform facilitates the online sale of dairy items, ensuring a seamless shopping experience for users.

  • Implemented user authentication and authorization for secure access to the platform.
  • Integrated a product catalog with categories for milk, cheese, yogurt, and other dairy products.
  • Utilized Django's ORM to manage product listings, inventory, and orders efficiently.
  • Designed a responsive and user-friendly UI using HTML, CSS, and Bootstrap framework.
  • Enabled payment gateway integration to process online transactions securely.

Technologies Used: Django, HTML, CSS, Bootstrap, JavaScript, PostgreSQL.

Project 3: Employee Management Web Application

Developed a simple mini project as a web application to handle employee details with add, delete, and update options. This application helps in managing employees within organizations efficiently.

Project 4: Investor Dashboard

Developed a comprehensive dashboard aimed at investors for visualizing and analyzing financial data, providing real-time insights and performance metrics. The project was built using Django, which significantly enhanced both the user interface (UI) and user experience (UX).

Technologies Used: Django, HTML, CSS, JavaScript, jQuery, AJAX, PostgreSQL,restfulapi.
